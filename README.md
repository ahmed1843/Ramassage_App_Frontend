# 🚛 Ramassage App - Frontend

Application web mobile de gestion et de signalement de ramassage de déchets.

## ✨ Fonctionnalités
- **Authentification** : Inscription et Connexion sécurisées (simulées via LocalStorage).
- **Signalement intelligent** :
  - Localisation automatique via **GPS/Géolocalisation**.
  - Prise de **photo** avec prévisualisation instantanée.
- **Notifications** : Alertes dynamiques pour le passage des camions et le suivi des signalements.
- **Profil Utilisateur** : Gestion des informations personnelles et des zones de ramassage.
- **Sécurité** : Système de "Gardien" (`auth-guard.js`) protégeant les pages privées.

## 🛠️ Technologies utilisées
- **HTML5 / CSS3** (Design Mobile-First & Écologique)
- **JavaScript (Vanilla)**
- **GitHub** (Versionnage)
